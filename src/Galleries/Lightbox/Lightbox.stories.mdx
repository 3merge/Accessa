## Custom open dialog

When given a function, the developer can customize the open
button. Since it's iterable, the first parameter will
contain properties of the active video.

<Preview>
  <Story name="Custom renderer">
    <LightboxWrapper lists={videos.splice(0, 1)}>
      {(video, renderer) => (
        <p style={{ position: 'relative', width: 330 }}>
          <img
            src={video.img}
            alt="Preview of video thumbnail"
            style={{ width: '100%' }}
          />
          <div
            style={{
              position: 'absolute',
              top: '50%',
              left: '50%',
              transform: 'translate(-50%,-50%)',
            }}
          >
            {renderer}
          </div>
        </p>
      )}
    </LightboxWrapper>
  </Story>
</Preview>
